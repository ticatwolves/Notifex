# Project Title

Notifex

## Description

End-2-End notification configuration and sending utility.

## Getting Started

### Dependencies

- GoLang 1.26+
- docker and docker compose

### Installing

- Clone `git clone https://github.com/sbshobhit/Notifex.git`
- Create and Update .env file
- docker-compose up 

### Executing program

- How to run the program
```
go mod download 
go run cmd/api/main.go
```
